#### 介绍

前端主要技术：Vue3、ELement-Plus、Vue-router、Pinia、Tailwind.css、Sass、Animate.css

后端主要技术：Node.js、Koa、Koa-router、Seqlize、Mysql

### 运行

#### blog-server  博客后端部分
进入到blog-server目录
pnpm i
pnpm run serve

#### blog-v3  博客前台展示
进入到blog-v3目录
pnpm i
pnpm run serve
#### blog-v3-admin  博客后台管理页面
进入到blog-v3-admin目录
pnpm i
pnpm run dev
